# Algorithmic Art

## Overview
Creates generative / algorithmic art using p5.js with seeded randomness and interactive parameter controls.

## When to trigger
- User asks for generative art, algorithmic art, flow fields, particle systems
- User wants art created with code

## Key rules
- Use seeded randomness so results are reproducible
- Expose key parameters for interactive exploration
- Never copy existing artists' styles — create original work
- Default canvas: 800x800, dark background unless specified

## Output format
Deliver a self-contained p5.js sketch (HTML + JS or .js file).
