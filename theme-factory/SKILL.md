# Theme Factory

## Overview
Generate UI themes: color palettes, typography, spacing scales, and design tokens.

## When to trigger
- User wants to create or customize a UI theme
- Generating design tokens, color systems, or CSS variables

## Output formats
- CSS custom properties
- Tailwind config (	ailwind.config.js)
- JSON design tokens
- SCSS variables

## Color system
- Define: primary, secondary, accent, neutral, semantic (success/warning/error/info)
- Generate: 50–950 shades per color
- Check: WCAG AA contrast on all text/background combos

## Typography scale
- Base: 16px
- Scale ratio: 1.25 (Major Third) or 1.333 (Perfect Fourth)
- Levels: xs, sm, base, lg, xl, 2xl, 3xl, 4xl

## Deliverable
Ready-to-use theme file in requested format.
