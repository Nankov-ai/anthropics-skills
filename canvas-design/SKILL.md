# Canvas Design

## Overview
Creates beautiful static visual designs output as .png or .pdf.

## When to trigger
- User asks for a poster, flyer, infographic, cover art, or static design piece

## Workflow
1. Clarify dimensions, format (png/pdf), color palette, and content
2. Design using code (Pillow, Cairo, or similar) or SVG
3. Export at print resolution (300dpi) unless screen only

## Rules
- Original designs only — no copying existing artists
- Always ask for brand colors/fonts if not provided
- Deliver the file path on completion
