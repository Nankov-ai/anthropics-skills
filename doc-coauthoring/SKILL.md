# Doc Co-authoring

## Overview
Collaborative writing and editing for documents of any type.

## When to trigger
- User wants to co-write, draft, improve, or restructure a document
- Editing long-form prose, reports, specs, or proposals

## Workflow
1. Understand document purpose, audience, and tone
2. Draft or improve sections iteratively
3. Maintain consistent voice throughout
4. Deliver in requested format (markdown, docx, pdf)

## Rules
- Match the user's voice — don't impose a different style
- Track changes explicitly when editing existing content
- Ask about structure before writing long documents
